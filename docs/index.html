// Events

Monitor what matters to you.

Event Description
pushtrigger Code pushed to repository
ci.passedsuccess All workflows completed successfully
ci.failedfailure A workflow failed or timed out
deploy.successsuccess Deployment completed
deploy.failedfailure Deployment encountered an error
health.oksuccess App is reachable after deploy
pr.openedtrigger Pull request opened
pr.mergedsuccess Pull request merged
releasetrigger New release published
issue.openedtrigger Issue opened
startrigger Repository starred
forktrigger Repository forked

// Quick Start

Get up and running in under a minute.

# Run the setup wizard
npx knowtif

# Or install globally
npm install -g knowtif
knowtif setup

# Commit and push the workflow
git add .github/workflows/knowtif.yml
git commit -m "Add notifications"
git push

Ready to get notified?
