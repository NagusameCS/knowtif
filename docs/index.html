Knowtif

Get notified when your GitHub changes propagate.
Push, CI, Deploy, Health Check - all in one command.

npx knowtif install

No server required. Runs entirely on GitHub Actions.

Features

Push Notifications

Know immediately when your code reaches GitHub.

CI Monitoring

Get notified when workflows pass or fail.

Deployment Status

Track deployments to any environment.

Health Checks

Verify your app is reachable after deploy.

PR Activity

Track pull requests, reviews, and merges.

Issue Tracking

Know when issues are opened or closed.

Notification Channels

Discord

Webhook integration with rich embeds

Pushover

Phone notifications with priority sounds

Email

SMTP with styled HTML emails

ntfy.sh

Free push for browser extensions

Webhook

Custom endpoint with HMAC signing

Events

Event Description
Push Received Your code has been received by GitHub
CI Passed All workflows completed successfully
CI Failed A workflow failed or timed out
Deployment Successful Your app was deployed
Deployment Failed Deployment encountered an error
Health Check Passed Your app is reachable after deploy
Health Check Failed App is not responding after deploy
New Pull Request A PR was opened
PR Merged A PR was merged
PR Approved Someone approved a PR
New Issue An issue was opened
New Release A release was published
New Star Someone starred your repo
Repository Forked Someone forked your repo

Quick Start

# Run the installer in your repo
npx knowtif install

# Follow the prompts to:
# 1. Choose events to monitor
# 2. Set up notification channels
# 3. Configure secrets

# Then commit and push
git add .github/workflows/knowtif.yml
git commit -m "Add Knowtif notifications"
git push
